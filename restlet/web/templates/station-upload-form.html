Options

<@forms.checkbox field=form.atts.dryRun>

If checked, no database modifications will be made.

<@forms.checkbox field=form.atts.allowUpdate>

If checked, this will allow updating stations. Otherwise, any existing stations will be ignored on ingest.

<@forms.checkbox field=form.atts.createOperators>

If checked, this will allow the ingester to define operators in the database if not defined already. Otherwise, a station with a missing operator will be ignored and a warning issued.
