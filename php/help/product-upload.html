Product Upload Instructions

Products for the website should be provided in a zip file. The name of the file is not important, but the contents must be named appropriately. The naming of the zip contents will mirror the products' type, process, and name. The zip file must have an internal structure two directories deep.

The top level directories in the zip file indicate the type, name, and process that are represented by all of the products beneath that directory. A given upload may contain multiple top level directories.

The product types and names listing can be found here.

The directories under the top level indicate the product names, specifically, the indicator, class, period, quadrant, and parameter the contents within.

Finally, the contents of the lowest directory should be the actual products. These should be named according to station name or station dataset id.
